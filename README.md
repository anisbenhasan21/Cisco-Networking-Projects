  # Cisco Packet Tracer - Networking Projects Portfolio

Welcome to my networking portfolio! This repository contains practical network topologies and configurations designed and simulated using **Cisco Packet Tracer**.

## 🛠️ Implemented Technologies & Protocols:
* **Routing Protocols:** Configuration of dynamic routing using **OSPF** (Open Shortest Path First) for efficient traffic paths.
* **VLAN Administration:** Implementing Virtual LANs to segment broadcast domains and enhance network security.
* **IP Management & Services:** Configuration of dynamic IP allocation using **DHCP Servers** and pool configurations.
* **Network Troubleshooting:** Verifying end-to-end connectivity via Cisco CLI using ping, traceroute, and routing table analysis.
